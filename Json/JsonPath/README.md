# Json Path

XML ning afzalliklaridan biri bu W3C standarti sifatida belgilangan XPathni o'z ichiga olgan ishlov berishning mavjudligi.
JSON uchun JSONPath deb nomlangan shunga o'xshash vosita paydo bo'ldi.
